# User Story 001

## Title

Start My Transformation Journey

### Story

As a new user,

I want Drona to understand my goals, lifestyle, and motivation,

So that I receive a personalized transformation plan instead of generic fitness advice.

### Acceptance Criteria

* Drona greets the user warmly.
* The conversation feels natural.
* The user can answer questions one at a time.
* The onboarding ends with a personalized plan and journey timeline.
